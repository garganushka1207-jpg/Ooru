# Ooru

> *The universe remembers what you leave behind.*

## About

Ooru is a reflective journaling experience built around a simple idea: memories don't have to live inside pages or folders.

Instead, every memory becomes a star in a personal night sky. Some are revisited when you're ready. Some remain as guiding lights. Others are set free when it's finally time to let them go.

Rather than organizing memories chronologically, Ooru encourages users to rediscover them naturally—like looking up at the night sky.

---

## Features

* Anonymous star-based memory storage
* Interactive night sky built with Three.js
* One memory assigned to one star
* A dedicated Pole Star for memories worth keeping
* Shooting star animation when a memory is released
* Local storage support
* Responsive glassmorphism interface

## Design Philosophy

Most journaling applications focus on organizing memories.

Ooru explores a different idea.

Instead of searching through entries, users wander through a personal universe where memories exist as stars. The experience is intentionally quiet, encouraging reflection rather than productivity.

The Pole Star preserves moments that continue to guide us, while the rest of the sky becomes a place where memories can either be revisited or released.
